# Lesson 02

To get started see the [Spring Quickstart](https://spring.io/quickstart).

## Start It

MacOS
```bash
./mvnw spring-boot:run
```

Windows
```bash
mvnw spring-boot:run
```

Browse Web Page
```
http://localhost:8080/hello
```

## Run Unit Tests

```bash
./mvnw test
```

## Run Feature Tets

```bash
./mvww -Pfeatures test
```
